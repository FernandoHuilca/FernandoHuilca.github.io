Fitec

Periodo: abril - agosto 2025

Descripción

FITEC es un sistema de gestión para una red de gimnasios con dos sucursales en Quito (Norte y Sur), implementando una base de datos distribuida que permite manejar información crítica como clientes, instructores, suscripciones y suplementos deportivos. Este proyecto fue desarrollado como parte de la materia de Bases de Datos Distribuidas, aplicando conceptos de fragmentación, replicación y consistencia en entornos distribuidos.

Tecnologías utilizadas

  • Java
  • SQLServer
  • GitHub
